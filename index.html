humanReadable (or just hr)

A command line tool to convert numbers to human readable forms

Description

hr is a tiny helper script meant to convert numbers in the output of another *nix command.

Requirements

hr currently depends on perl.

Installation Instructions

hr is implemented as a perl script

You can install it by linking or copying hr.pl as hr in a directory in your path.

Usage

$ hr --help
USAGE: hr [column] [current units] [files]
* column - column to transform
* current units - if units are already in a unit: specify with K, M, G, T, P, E

# Examples

Simulate the -lh option of ls:

ls -l | hr 4


sort by size:

ls -l | sort --key=4n | hr 4


handles multiple columns, and when some columns are in different starting units

df | hr 2 | hr 3 | hr 4


handles multiple columns, even when some columns are in different starting units

df -k | hr 2 | hr 3 K | hr 4 k
